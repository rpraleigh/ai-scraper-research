# Gemini 2.5 Pro Site Researcher

A deep-crawling web researcher built with **Crawl4AI**, **Streamlit**, and **Gemini 2.5 Pro**.

## Setup
1. Clone the repo.
2. Create a `.env` file with your `GOOGLE_API_KEY`.
3. Install dependencies: `pip install -r requirements.txt`
4. Install Playwright browsers: `python -m playwright install chromium`
5. Run the app: `streamlit run app.py`
